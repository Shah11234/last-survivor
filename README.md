# Last Survivor 🗡️

A 2D platformer game built with Godot 4 as part of my game development portfolio.

## 🎮 Gameplay
- Side-scrolling 2D platformer
- Player combat with enemy slimes
- Coin collection system
- Kill zones and platform mechanics
- Background music and sound effects

## 🛠️ Built With
- [Godot Engine 4.6](https://godotengine.org/) 
- GDScript
- Custom tilemap and sprite assets

## 🚀 How to Run
1. Download and install [Godot 4](https://godotengine.org/download)
2. Clone this repo:
```
   git clone https://github.com/Shah11234/last-survivor.git
```
3. Open Godot → Import Project → select the `project.godot` file
4. Press F5 to play

## 📁 Project Structure
```
last-survivor/
├── assets/        # Sprites, sounds, music, fonts
├── scenes/        # All .tscn scene files
├── scripts/       # All .gd GDScript files
└── project.godot  # Godot project file
```
